# Configuração da API do Gemini

## Como obter a chave da API gratuita do Google Gemini

1. Acesse: https://makersuite.google.com/app/apikey
2. Faça login com sua conta Google
3. Clique em "Create API Key" ou "Get API Key"
4. Copie a chave gerada

## Como configurar no código

1. Abra o arquivo: `assets/js/demo-cliente.js`
2. Procure pela linha que contém: `const API_KEY = 'SUA_API_KEY_AQUI';`
3. Substitua `SUA_API_KEY_AQUI` pela sua chave da API
4. Salve o arquivo

## Funcionalidades implementadas

✅ Integração com API do Gemini
✅ Validação de nicho (apenas programação)
✅ Alerta para perguntas fora do nicho
✅ Créditos ao Google Gemini e Grupo PIM LA10
✅ Respostas especializadas em programação

## Limites da API gratuita

A API gratuita do Gemini tem limites de uso. Consulte a documentação oficial para mais detalhes.

## Segurança

⚠️ **IMPORTANTE**: Não compartilhe sua chave da API publicamente. Em produção, considere usar um backend para proteger a chave.
